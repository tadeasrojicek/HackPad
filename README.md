# HackPad - Custom 3-Key Macropad

## Description
A custom 3-key macropad with 6 RGB LEDs built for Hack Club Hackpad.

## Bill of Materials
| Part | Quantity |
|------|----------|
| Seeed XIAO RP2040 | 1 |
| MX switches | 3 |
| SK6812 MINI-E RGB LEDs | 6 |
| 1N4148 diodes | 3 |

## Firmware
KMK on CircuitPython - 3 keys with RGB LEDs
