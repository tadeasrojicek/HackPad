# HackPad - Custom 3-Key Macropad

## Description
I tried to make an working Hackpad. It is my first time doing something like this, so hopeffuly no error s;)

## Bill of Materials
| Part | Quantity |
|------|----------|
| Seeed XIAO RP2040 | 1 |
| MX switches | 3 |
| SK6812 MINI-E RGB LEDs | 6 |

## Firmware
KMK on CircuitPython - 3 keys with RGB LEDs
